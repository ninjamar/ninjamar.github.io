You will need nasm and qemu to use this tutorial


Make a file called boot.asm


      main:
        jmp main
      times 510-($-$$) db 0
      dw 0xaa55

We need to compile this


      cd path/to/boot.asm
      nasm -f bin boot.asm -o boot.bin
      qemu-system-x86_64 boot.bin

A display window should open. It should be stuck in a infinite loop.Lets go through this line by line


      main:
        jmp main

This makes a function called main that loops forever


        times 510-($-$$) db 0

This fills the rest of the file with 0 because the max size of the bootloader is 512


        dw 0xaa55

That is some sort of magic byte thing
